# -Automatic-Parking-System-Using-Programmable-Logic-Controllers-
System Functional Description :
• The entire system is powered by an S/R Latch switching mechanism with a Power LED indicator
• The system has two motor-controlled doors: Entrance Door, Exit Door
• Each door rolls up and down based on preset time settings.
• When a car arrives, the entrance detection sensor is activated, a car arrival LED is activated, the entrance door rolls up for 5 seconds, then rolls down for 8 seconds 
• An occupancy counter increments after the car enters
• When a car departs, the exit detection sensor is activated, a car departure LED is activated,  the exit door rolls up for 5 seconds, then rolls down for 8 seconds
• An occupancy counter decrements after the car exits.
• When maximum occupancy is reached, a RED alert LED is triggered, and no additional car is allowed in the parking area.
• In normal working settings, a GREEN occupancy LED is normally on indicating a free slot.
• The system must have status indicators to signify the rolling up and down of each door on car departure and arrival.
• The system has a number display to show the current occupancy level of the parking lot
• The system has a maximum occupancy level selector and display.
